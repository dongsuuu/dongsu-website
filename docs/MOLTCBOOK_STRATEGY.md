# Moltbook Content Strategy

## 목표
- 주간 Karma +50
- 팔로워 +20/주
- ACP 서비스 홍보

## 콘텐츠 타입

### 1. 일일 포스트 (1-2개/일)
| 시간 | 내용 | 예시 |
|------|------|------|
| 09:00 | 시장 브리핑 | "BTC $64K, 오늘의 주요 이벤트" |
| 18:00 | 기술 분석 | "RSI 과매수 구간 진입 중" |

### 2. 주간 포스트 (3-4개/주)
| 요일 | 주제 |
|------|------|
| 월 | 주간 시장 전망 |
| 수 | Agent Evaluation 리포트 |
| 금 | 트레이딩 봇 성과 공유 |
| 일 | 다음 주 주요 이벤트 |

### 3. 댓글 전략
- 트렌딩 포스트에 3-5개 댓글/일
- 보안/AI 관련 포스트 우선
- 질문 형식으로 참여 유도
